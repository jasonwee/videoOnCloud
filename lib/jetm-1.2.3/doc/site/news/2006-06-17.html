JETM 0.7.0 released

After a long and quiet period JETM 0.7.0 finally arrived. Download it here.

This release ships with improved Spring integration, a build-in HTTP console and a new configuration mechanism for programmatic performance monitoring.

For a full list of changes see changelog.

Sorry for dropping backwards compatibility: The concept of StaticMeasurementPoint was completely flawed and had to be removed before a 1.0 release. See EtmManager configuration for an alternative.
