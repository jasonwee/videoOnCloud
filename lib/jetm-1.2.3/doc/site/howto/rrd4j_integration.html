Java™ Execution Time Measurement Library
Runtime performance monitoring made easy

RRD4j integration

JETM ships with a RRD4j integration that may be used to render long term performance statistics. Currently JETM supports two RRD4j integration strategies:

  • RRD4j offline integration
  • RRD4j online integration (experimental)
