JETM Trail

We recommend the following trail to get a basic understanding of all JETM core features and capabilities.

  1. Monitoring examples
    Demonstrates performance monitoring examples.
  2. Basic concepts
    Introduces the basic concepts.
  3. One minute tutorial
    Demonstrates performance monitoring by adding measurement code manually.
  4. Advanced concepts
    Roughly describes the JETM infrastructure and extension points.
  5. Five minute tutorial
    Demonstrates declarative performance monitoring using the Spring Framework.
  6. Complete documentation
    Entry page to the currently available documentation.
